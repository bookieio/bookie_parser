Bookie  - bookmark your web

Welcome to the Bookie Readable Parser. This site provides an api for parsing the content of a web page and providing a clear and readable version of it.

Fork me on Github

In the future we plan on adding additional logic to the application to help aid in selecting tags and other bits of metadata about the content.

Try it out

Bookmarklet

Drag this bookmarklet to your browser's bookmark bar and click on it when viewing a site you want to read in a clean way.

Read with Bookie 
javascript:(function() {location.href='https://readable.bmark.us/view/'+encodeURIComponent(location.href)}())

API

Example calls are shows using the HTTPie Python tool. It really makes testing out api calls and such a lot easer.

Posting Content

$ http -f POST "http://readable.bmark.us/readable/google.com" content="<h1>Hello world</h1>
        HTTP/1.1 200 OK
        Content-Type: application/json; charset=UTF-8
        Server: TornadoServer/2.2 (Gunicorn/0.14.2)
        Content-Length: 115
        Connection: keep-alive

        {
            "readable": "<body id=\"readabilityBody\"><h1>Hello world</h1></body>",
            "title": "Unknown",
            "url": "google.com"
        }

Fetching URL Content

$ http -f GET "http://readable.bmark.us/readable/tornadoweb.org%2Fdocumentation%2Fhttpclient.html%23tornado.httpclient.AsyncHTTPClient.configure"

        HTTP/1.1 200 OK

        Server: TornadoServer/2.2 (Gunicorn/0.14.2)
        Content-Length: 4812

Viewing URL Content

$ http -f GET "http://readable.bmark.us/view/tornadoweb.org%2Fdocumentation%2Fhttpclient.html%23tornado.httpclient.AsyncHTTPClient.configure"
